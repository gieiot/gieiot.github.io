Remarks
 
T must be an struct with attribute [StructLayout(LayoutKind.Sequential, Pack = 1)] applied. 
Allowed struct fields are some primitives: Boolean, Byte, SByte, Int16, UInt16, Int32, UInt32, Int64, UInt64, Char, Single, Double; strings, one dimension arrays and other sequentially layouted structs. 
 
Boolean fields must have attribute [MarshalAs(UnmanagedType.UnmanagedType.I1)] applied. 
String fields must have attribute [MarshalAs(UnmanagedType.ByValTStr, SizeConst = xx)] applied. 
Array fields must have attribute [MarshalAs(UnmanagedType.ByValArray, SizeConst = x)] applied. Elements could be same primitive types or other sequentially layouted structs. 
Array fields of boolean elements must have attribute [MarshalAs(UnmanagedType.ByValArray, ArraySubType = UnmanagedType.I1, SizeConst = xx )] applied. 
 
A concrete class is derived from this generic abstract class as: DerivedClass : IotDeviceData<T_STRUCT>  
T_STRUCT will be accesibles as _data private member in devided class.  
 
To create an object of the derived class from a byte list, a constructor calling the parameterized base constructor is required: 
public DerivedClass(List<byte> byteList) : base(byteList) {}
